About the plugin

ClipSwitch is a compact Unity Editor workbench for previewing, replacing, comparing, and batch-processing AudioClips while preserving GUIDs and existing project references.

Features

  • Waveform previews with instant playback from any position
  • Safe file replacement that keeps GUIDs and scene or prefab references intact
  • Automatic history snapshots before destructive operations
  • Batch clip selection and side-by-side comparison workflow
  • Built-in processing for trim, gain, normalize, reverse, fade, and pitch or speed tasks

Installation

Open the latest published GitHub release, download the package archive, extract it, then use Add package from disk in Unity Package Manager and select package.json.
