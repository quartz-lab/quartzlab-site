# Contributing

Issues and pull requests in this repository should concern the QuartzLab website, generator, documentation pipeline, or deployment configuration. Report a bug in a specific Unity plugin in that plugin's own repository, linked from its catalog page.

Before opening a pull request:

1. Use Node.js 22 or newer.
2. Run `npm run check`.
3. Keep the existing static RU/EN architecture and production base path.
4. Do not commit `_site/` or other generated artifacts.
5. Do not add tokens, `.env` files, credentials, or personal contact data.

Keep changes focused and explain any generated-output or routing impact in the pull request.
